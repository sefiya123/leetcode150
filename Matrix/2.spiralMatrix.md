54. Spiral Matrix

Given an m x n matrix, return all elements of the matrix in spiral order.

 

Example 1:


Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [1,2,3,6,9,8,7,4,5]


```cpp
    class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int>ans;
        int n = matrix.size();
        int m = matrix[0].size();
        int left = 0;
        int top =0;
        int right = m-1;
        int bottom = n-1;

        while(left<=right && top<=bottom) {
            for(int i = left; i<=right;i++) {
                ans.push_back(matrix[top][i]);
            }
            top++;
            for(int i = top; i<=bottom;i++) {
                ans.push_back(matrix[i][right]);
            }
            right--;
            if (left>right || top>bottom) break;
            for(int i = right; i>=left;i--) {
                ans.push_back(matrix[bottom][i]);
            }
            bottom--;
            for(int i = bottom; i>=top;i--) {
                ans.push_back(matrix[i][left]);
            }
            left++;

        }
        return ans;
    }
};

🕒 Time Complexity (TC):
Let:

n = number of rows

m = number of columns
Then the total number of elements = n * m.

Each element is visited exactly once.

So the total work done is proportional to the number of elements.

✅ Time Complexity = O(n * m)

💾 Space Complexity (SC):
You create an output vector ans that stores all the n * m elements.

Other than that, only a few integer variables (left, right, top, bottom, i) are used.

✅ Space Complexity = O(n * m) (for the output)

If you're allowed to exclude output space, then the auxiliary SC = O(1), i.e., constant extra space.

🔁 Summary:
Metric	Value
Time Complexity	O(n * m)
Space Complexity	O(n * m) (output) or O(1) auxiliary
